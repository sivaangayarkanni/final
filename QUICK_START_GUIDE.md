### Task Management

#### Accessing Tasks
**Navigation:** Sidebar > Tasks (Calendar icon)

#### Creating a Task
1. Go to **Tasks** page
2. Click **"Add Task"** button
3. Fill in:
   - **Title** (required)
   - **Description** (optional)
   - **Due Date** (required)
   - **Priority** (Low/Medium/High)
   - **Category** (Follow-up/Meeting/Call/Email/Other)
   - **Related To** (optional) - Link to contact/lead
4. Click **"Add Task"**

#### Managing Tasks
- **Complete:** Click the checkbox icon
- **Edit:** Click the edit (pencil) icon
- **Delete:** Click the delete (trash) icon

#### Filtering Tasks
Use the dropdown to filter by:
- All Tasks
- Pending
- Completed
- High/Medium/Low Priority
- Follow-ups/Meetings/Calls

#### Searching Tasks
Use the search bar to find tasks by title or description

## ğŸ”§ Troubleshooting

### Tasks Not Showing?
- Check your filters (might be set to "Completed")

### Reminders Not Appearing?
